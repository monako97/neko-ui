## API

| 属性     | 说明                                 | 类型    | 默认值 | 版本 |
| -------- | ------------------------------------ | ------- | ------ | ---- |
| class    | 自定义类名                           | string  | -      | -    |
| css      | 自定义样式表                         | string  | -      | -    |
| offset   | 查看时的位置                         | number  | 0      | -    |
| dots     | 开启指示器                           | boolean | -      | -    |
| autoplay | 设置自动播放时长, 不设置时不自动播放 | number  | -      | -    |
