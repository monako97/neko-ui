# 水波纹

> 波纹溢出动画, 用于标记用户在线状态

## 直接在 mdx 中写

import { WaveCircle } from 'neko-ui';

<div style={{ display: 'flex', height: 300 }}>
  <WaveCircle bgColor="pink" />
  <WaveCircle>默认</WaveCircle>
</div>

```tsx
import { WaveCircle } from 'neko-ui';

ReactDOM.render(<WaveCircle bgColor="pink" />, document.body);
```
