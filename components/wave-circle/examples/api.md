## API

组件的属性说明如下：

| 属性           | 说明       | 类型            | 默认值                      |
| -------------- | ---------- | --------------- | --------------------------- |
| bgColor        | 背景颜色   | `string`          | `@primary-color`              |
| timingFunction | 动画曲线   | `string`          | `@transition-timing-function` |
| className      | 自定义类名 | `string`          | -                           |
| children       | 子元素     | `React.ReactNode` | -                           |
