在输入框上添加前缀或后缀图标
通过 `suffix` 属性给输入框尾部添加内容
通过 `prefix` 属性给输入框头部添加内容

```tsx
<div style={{ display: 'flex', flexDirection: 'column', gap: 12 }}>
  <Input label="用户" placeholder="请输入名称" prefix="👨" defaultValue="admin" />
  <Input label="密码" type="password" placeholder="请输入密码" prefix="㊙️" />
  <Input label="验证码" autoComplete="one-time-code" placeholder="请输入验证码" suffix="45秒" disabled />
</div>
```
