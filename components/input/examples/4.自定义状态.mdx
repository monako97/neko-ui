使用 `status` 为输入框添加状态，可选 `error`、`warning`、`success`

```tsx
<div style={{ display: 'flex', flexDirection: 'column', gap: 12 }}>
  <Input type="email" label="Email" disabled status="success" />
  <Input type="password" label="Password" status="error" />
  <Input type="tel" label="Tel" status="warning" />
  <Input type="user" label="User" status="success" />
</div>
```
