输入框内置有大、中、小三种尺寸。
通过设置 `size` 为 `large`、`small` 分别把尺寸设为大、小尺寸。若不设置 `size`，则尺寸为中 `normal`。

```tsx
<div style={{ display: 'flex', flexDirection: 'column', gap: 12 }}>
  <Input label="小尺寸" size="small" />
  <Input label="默认大小" placeholder="请输入文字" />
  <Input label="大尺寸" size="large" placeholder="请输入文字" />
</div>
```
