## API

| 属性   | 说明         | 类型    | 默认值 | 版本 |
| ------ | ------------ | ------- | ------ | ---- |
| class  | 自定义类名   | string  | -      | -    |
| css    | 自定义样式表 | string  | -      | -    |
| rows   | 自定义行数   | number  | 3      | -    |
| active | 显示动画     | boolean | false  | -    |
| avatar | 显示头像     | boolean | false  | -    |
| title  | 显示标题     | boolean | false  | -    |
