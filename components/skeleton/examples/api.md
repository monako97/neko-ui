## API

| 属性      | 说明         | 类型                  | 默认值 | 版本 |
| --------- | ------------ | --------------------- | ------ | ---- |
| className | 自定义类名   | string                | -      | -    |
| style     | 自定义样式表 | `React.CSSProperties` | -      | -    |
| rows      | 自定义行数   | number                | 3      | -    |
| active    | 显示动画     | boolean               | false  | -    |
| avatar    | 显示头像     | boolean               | false  | -    |
| title     | 显示标题     | boolean               | false  | -    |
