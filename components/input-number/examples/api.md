## API

| 属性     | 说明                     | 类型                       | 默认值 | 版本 |
| -------- | ------------------------ | -------------------------- | ------ | ---- |
| value    | 值                       | `number`                   | -      | -    |
| min      | 最小值                   | `number`                   | -      | -    |
| max      | 最大值                   | `number`                   | -      | -    |
| onChange | 值变更时触发的函数       | `(value?: number) => void` | -      | -    |
| step     | 每次改变步数，可以为小数 | `number`                   | -      | -    |
