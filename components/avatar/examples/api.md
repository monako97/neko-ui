## API

| 属性     | 说明         | 类型                                     | 默认值 | 版本 |
| -------- | ------------ | ---------------------------------------- | ------ | ---- |
| class    | 自定义类名   | string                                   | -      | -    |
| css      | 自定义样式表 | `string`                                 | -      | -    |
| src      | 头像         | string                                   | -      | -    |
| username | 用户名       | string                                   | -      | -    |
| alt      | 替代文本     | string                                   | -      | -    |
| color    | 背景颜色     | string                                   | -      | -    |
| size     | 尺寸         | number \| `small` \| `normal` \| `large` | -      | -    |
