## API

| 属性         | 说明                   | 类型                  | 默认值 | 版本 |
| ------------ | ---------------------- | --------------------- | ------ | ---- |
| src          | 图片地址               | string                | -      | -    |
| alt          | 图片 alt               | string                | -      | -    |
| open         | 大图查看               | boolean               | -      | -    |
| onOpenChange | 开启关闭大图的回调函数 | (open: boolean): void | -      | -    |
| maskClosable | 点击遮罩关闭           | boolean               | false  | -    |
| escClosable  | 通过 `esc` 按键关闭    | boolean               | true   | -    |
