# 代码高亮与编辑

> 轻量级且高性能的代码高亮和编辑组件，旨在提升开发者的编码体验。

支持的语言类型包括：支持同时指定多个语言

|                |                 |                |
| :------------: | :-------------: | :------------: |
|    **bash**    |    **json**     |    **rss**     |
|   **shell**    | **webmanifest** |   **regex**    |
|     **sh**     |     **jsx**     |    **rust**    |
|   **clike**    |     **tsx**     |    **sql**     |
|    **css**     |    **less**     |   **swift**    |
|   **docker**   |    **html**     |    **toml**    |
| **dockerfile** |   **mathml**    | **typescript** |
|    **git**     |     **svg**     |     **ts**     |
|    **diff**    |     **xml**     |    **yaml**    |
| **javascript** |    **ssml**     |    **yml**     |
|     **js**     |    **atom**     |   **matlab**   |

## 注册组件 `只需注册一次`

```js
import { Code, registry } 'neko-ui';

registry(Code);
```
