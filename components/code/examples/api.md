## API

| 属性        | 说明             | 类型    | 默认值 | 版本 |
| ----------- | ---------------- | ------- | ------ | ---- |
| css         | 自定义样式表     | string  | -      | -    |
| code        | 内容             | string  | -      | -    |
| line-number | 显示代码行号     | boolean | -      | -    |
| toolbar     | 开启代码块工具条 | boolean | -      | -    |
| edit        | 支持编辑         | boolean | -      | -    |
