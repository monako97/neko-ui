## API

| 属性               | 说明             | 类型                | 默认值       | 版本 |
| ------------------ | ---------------- | ------------------- | ------------ | ---- |
| className          | 自定义类名       | string              | -            | -    |
| style              | 自定义样式表     | React.CSSProperties | -            | -    |
| code               | 内容             | string              | -            | -    |
| lineNumber         | 显示代码行号     | boolean             | false        | -    |
| tools              | 开启代码块工具条 | ToolType[]          | ['copy']     | -    |

## ToolType

- copy
