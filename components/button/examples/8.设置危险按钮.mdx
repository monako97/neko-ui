添加 `danger` 属性将按钮标记为危险操作

```jsx
<div style={{ display: 'flex', gap: 8, flexWrap: 'wrap' }}>
  <Button danger>default</Button>
  <Button fill danger>
    fill
  </Button>
  <Button dashed danger>
    dashed
  </Button>
  <Button ghost danger>
    ghost
  </Button>
  <Button flat danger>
    flat
  </Button>
  <Button link danger>
    link
  </Button>
</div>
```
