## API

| 属性      | 说明         | 类型                                                        | 默认值    | 版本   |
| --------- | ------------ | ----------------------------------------------------------- | --------- | ------ |
| className | 自定义类名   | string                                                      | -         | -      |
| style     | 自定义样式表 | `React.CSSProperties`                                       | -         | -      |
| type      | 按钮类型     | `success` \| `error` \| `primary` \| `warning` \| `default` | `default` | -      |
| infinite  | 无限动画     | boolean                                                     | -         | -      |
| ghost     | 透明背景     | boolean                                                     | -         | -      |
| fill      | 实色背景     | boolean                                                     | -         | -      |
| circle    | 圆形按钮     | boolean                                                     | -         | -      |
| dashed    | 虚线按钮     | boolean                                                     | -         | -      |
| flat      | 扁平按钮     | boolean                                                     | -         | -      |
| disabled  | 禁用按钮     | boolean                                                     | -         | -      |
| link      | 链接按钮     | boolean                                                     | -         | -      |
| block     | 块按钮       | boolean                                                     | -         | -      |
| size      | 尺寸         | `small` \| `normal` \| `large`                              | `normal`  | 1.0.36 |
| danger    | 危险按钮     | boolean                                                     | -         | -      |
