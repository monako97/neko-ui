## API

| 属性     | 说明     | 类型                                                                    | 默认值 | 版本 |
| -------- | -------- | ----------------------------------------------------------------------- | ------ | ---- |
| type     | 按钮类型 | `success` \| `error` \| `danger` \| `primary` \| `warning` \| `default` | -      | -    |
| infinite | 无限动画 | boolean                                                                 | -      | -    |
| ghost    | 透明背景 | boolean                                                                 | -      | -    |
| fill     | 实色背景 | boolean                                                                 | -      | -    |
| circle   | 圆形按钮 | boolean                                                                 | -      | -    |
| dashed   | 虚线按钮 | boolean                                                                 | -      | -    |
| float    | 文字按钮 | boolean                                                                 | -      | -    |
| disabled | 禁用按钮 | boolean                                                                 | -      | -    |
| link     | 链接按钮 | boolean                                                                 | -      | -    |
