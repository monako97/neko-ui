添加 `flat` 属性将其设置为扁平按钮。

```jsx
<div style={{ display: 'flex', gap: 8, flexWrap: 'wrap' }}>
  <Button flat>按钮</Button>
  <Button type="primary" flat>
    按钮
  </Button>
  <Button type="success" flat>
    按钮
  </Button>
  <Button type="warning" flat>
    按钮
  </Button>
  <Button type="error" flat>
    按钮
  </Button>
</div>
```
