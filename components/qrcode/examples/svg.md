---
title: 自定义二维码渲染方式
description: 设置二维码的渲染方式，支持 SVG 或 Canvas。
---

```html
<n-qrcode value="draw svg" type="svg" style="margin:20px"></n-qrcode>
<n-qrcode value="draw canvas" type="canvas"></n-qrcode>
```

```jsx
<n-qrcode value="draw svg" type="svg" style={{ margin: '20px' }} />
<n-qrcode value="draw canvas" type="canvas" />
```
