将 `trigger` 为 `contextMenu`，将可以通过右键触发提示框显示

```tsx
<Popover
  arrow
  content="右键触发"
  trigger="contextMenu"
  style={{
    width: '100%',
    height: 150,
    backgroundColor: 'var(--disable-bg)',
    borderRadius: 8,
    lineHeight: '150px',
    textAlign: 'center',
    fontSize: 'x-large',
  }}
>
  <div
    style={{
      opacity: 0.4,
    }}
  >
    右键点我
  </div>
</Popover>
```
