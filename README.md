# Neko UI

🐾 🐾 🐾 🐾

> 这是一个基于 SolidJS、Typescript 开发的适用于任何技术栈的 WebComponents 组件库, 你可以在其他任意 web 框架中像 `div` 一样使用它

## 安装

```shell
npm install neko-ui -S
# or
yarn add neko-ui -S
```

### 使用

```html
<n-button>按钮</n-button>
```

## 在 html5 中使用

```html
<!-- 引入 CDN 资源 -->
<script src="https://cdn.jsdelivr.net/npm/neko-ui/lib/index.js"></script>
<!-- 使用 -->
<body>
  <n-button>按钮</n-button>
</body>
```
