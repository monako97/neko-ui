# neko-ui

> Neko UI


## 使用

```shell
yarn add neko-ui -S
```

```tsx
import React from 'react';
import { Markdown } from 'neko-ui';

const Demo = () => {
  return <Markdown text={`### h1 \n > csacas`} />;
};

export default Demo;
```

## 安装依赖

```shell
yarn install
```

## 启动项目
```shell
yarn start
```

## 打包项目
```shell
yarn build
```
