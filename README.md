## 安装

```shell
npm install neko-ui -S
# or
yarn add neko-ui -S
```

### 使用

```html
<n-button>按钮</n-button>
```

## 在 html5 中使用

```html
<!-- 引入 CDN 资源 -->
<script src="https://cdn.jsdelivr.net/npm/neko-ui@2.0.1/lib/index.js"></script>
<!-- 使用 -->
<body>
  <n-button>按钮</n-button>
</body>
```
