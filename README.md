## 安装 (npm)

```shell
npm install neko-ui -S
# or
yarn add neko-ui -S
```

### 使用 (npm)

```html
<n-button>按钮</n-button>
```

## 安装 (umd)

```html
<script src="https://cdn.staticaly.com/gh/monako97/cdn/main/npm/neko-ui/2.0.1/lib/index.js"></script>
```

### 使用 (umd)
